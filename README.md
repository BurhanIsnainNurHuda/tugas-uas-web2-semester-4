E-Inventory System (UAS Web 2)
Sistem manajemen inventaris barang full-stack yang mengimplementasikan Decoupled Architecture untuk memisahkan API service dan client-side.

 Tech Stack
Backend: CodeIgniter 4 (RESTful API, Resource Controller, Token-based Auth).

Frontend: VueJS 3, Vue Router (SPA), Axios (HTTP Client), TailwindCSS (Utility-first UI).

Database: MySQL/MariaDB.

 Arsitektur Sistem
Sistem ini menggunakan alur data asinkronus antara frontend dan backend.# tugas-uas-web2-semester-4

Fitur Utama
Authentication: Implementasi Token-based security (Bearer Token) pada setiap request data.

REST API: Standarisasi respons API menggunakan format JSON.

Dynamic UI: Rendering data barang real-time tanpa page refresh (SPA).

Access Control: Pembatasan akses endpoint via AuthFilter.

Instalasi & Cara Jalankan
Backend:

Masuk ke direktori backend-api.

Konfigurasi database di file .env.

Jalankan server: php spark serve.

Frontend:

Gunakan ekstensi Live Server di VS Code pada file index.html di dalam folder frontend-spa.

Pastikan URL base di app.js sudah mengarah ke http://localhost:8080/api.
